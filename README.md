# FinTech – Personal Finance Dashboard

FinTech is a web-based personal finance management system that helps users track income, monitor expenses, set savings goals, and receive financial suggestions.
The application provides a modern dashboard interface for analyzing financial activity and improving money management.

This project was developed as a **college project** using core web technologies.

---

## Features

### Dashboard

The dashboard provides a financial overview including:

* Monthly Income
* Total Expenses
* Net Savings
* Savings Rate

It also includes:

* Recent expenses table
* Spending breakdown visualization

---

### Expense Tracking

Users can:

* Add new expenses
* Categorize transactions
* View expense history
* Monitor spending behavior

Expense categories include:

* Food
* Transport
* Shopping
* Bills
* Entertainment
* Other

---

### Savings Goals

Users can create financial targets and track progress.

Features include:

* Creating savings goals
* Setting target amounts
* Tracking progress toward financial objectives

---

### AI Advisory

The AI advisory system analyzes financial inputs and generates simple financial suggestions based on:

* Total savings
* Financial goals
* Risk tolerance
* Investment time horizon

This helps users plan smarter financial decisions.

---

### User Profile

The profile section allows users to:

* View personal details
* Manage monthly income
* Update financial information

---

## Usage

1. Create an account or login.
2. Add your monthly income.
3. Track expenses using the expense tracker.
4. Set savings goals to plan your finances.
5. View insights on the dashboard.
6. Use AI Advisory for financial suggestions.

---

## Currency

All financial values are displayed in **Indian Rupees (₹)**.
