FEATURES

1. USER AUTHENTICATION

Signup Page:
Fields:
Full Name
Email
Password
Monthly Salary

Login Page:
Email
Password

API Routes:

POST /signup
POST /login

Passwords must be stored in the database.

---

2. DASHBOARD PAGE

Display financial overview cards:

Monthly Salary
Total Expenses
Net Savings
Savings Rate %

Formula:

Net Savings = Salary - Total Expenses
Savings Rate = (Net Savings / Salary) × 100

Also display a table of recent expenses.

---

3. EXPENSE TRACKER

Allow users to add expenses.

Fields:

Description
Category
Amount
Date

Categories:

Food
Transport
Shopping
Bills
Entertainment
Other

APIs:

GET /expenses
POST /expenses

Expenses must be linked to the logged-in user.

---

4. SAVINGS GOALS

User can create savings goals.

Fields:

Goal Name
Target Amount
Months to Achieve Goal

The Target Amount input must allow:

typing numbers manually
arrow increment/decrement

Display:

Monthly saving required

Formula:

Monthly Saving = Target Amount / Months

---

5. INVESTMENT ADVISORY PAGE

Create a simple advisory section.

Title:

AI Spending Advice

Description:

Provide financial tips based on expenses.

Example logic:

If food spending > 30% of salary
show suggestion to reduce food expenses.

---

6. PROFILE PAGE

Display:

User name
Email
Monthly salary

Allow updating salary.

API:

PUT /salary
